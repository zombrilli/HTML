Introdução

< Voltar

HTML é o formato que define a estrutura onde as páginas são elaboradas.

A estrutura básica de um HTML é:

<html>

<head> carrega antes de abrir pro usuário </head>

<body> Aparece para o usuário </body>

</html>

Tags

são elementos HTML utilizados para informar qual o tipo de estrutura está sendo construída

    Exemplos de Tags

  • <title> título da página
  • <i> itálico
  • <strong> negrito
  • <input>
  • <img>

Atributos

São palvras especias usadas dentro de tags para controlar o comportamento do elemento

    Exemplos de atributos

  • id - identificador
  • style - CSS, ex: color
  • class - padronizar

    • input

  • type - tipo de entrada

    • type="atributo"

  • text - campo de texto
  • number - campo numérico
  • colors - seletor de cor

    • img

  • src - fonte da imagem/li>
  • width - tamanho da imagem

  • br - quebra linha

Configurar Links

Os links podem ser configurados usando a tag <a>
Essa tag permite a ancoragem para o link a ser seguido

    Atributos para links
  • href - caminho a ser seguido
  • href="#'identificador' - aconragem na mesma página
  • target
    • _blank - abre em uma nova aba
    • _selg - abre na mesma aba
    >
  • titlle - texto ao clicar no link
